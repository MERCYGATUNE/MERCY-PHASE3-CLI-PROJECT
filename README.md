# Description
This is a simple CLI application to manage restaurant reservations. The application uses SQLite for data persistence.

## Setup

1. Install dependencies using Pipenv:
    ```sh
    pipenv install
    ```

2. Activate the virtual environment:
    ```sh
    pipenv shell
    ```

## Usage

Run the CLI:
```sh
python lib/cli.py
